<h1>구현한 기능</h1>

```


해당 기능들을 구현하였습니다.

* 쓰레기 랜덤 생성
* 올바른 분리수거 판단 기능
* 쓰레기 토스 중력 가속도 계산

* 쓰레기 분리수거 앱 및 게임 연동
* 쓰레기 분리수거 게임에서 발생하는 Log 데이터 감시 모니터링 구현
```
